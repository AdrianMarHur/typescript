# TypeScript Exercises Repository

This repository contains a collection of TypeScript modules created to practice the language fundamentals and progressively apply more advanced typing features.

The focus of this project is to understand how TypeScript improves code safety, structure, and maintainability compared to plain JavaScript.

---

## 📁 Project Structure

typescript/
├── modulo-1/
├── modulo-2/

Each module is independent and targets a specific learning objective.

---

## 📘 Module 1: TypeScript Fundamentals

Introduction to the core features of TypeScript.

Main concepts:
- Basic variable and function typing.
- Primitive types (`string`, `number`, `boolean`).
- Typed objects and interfaces.
- Type inference.
- TypeScript configuration (`tsconfig.json`).
- Compilation workflow from TypeScript to JavaScript.

This module uses simple examples to focus on learning the language syntax and behavior.

---

## 📙 Module 2: Advanced Typing and Code Structure

Application of more advanced TypeScript features to write safer and more structured code.

Main concepts:
- Domain modeling using `interface` and `type`.
- Discriminated Unions for state control.
- Exhaustive checks using the `never` type.
- Generic functions and utilities (`<T>`).
- Typed asynchronous logic.
- Code organization and separation of concerns.

This module demonstrates how TypeScript helps detect errors at compile time and improves code robustness.
